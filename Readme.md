# TDC-Simulacion

Pequeño proyecto que contiene un archivo HTML de simulación.

## Ejecutar

- Abre el archivo `Simulacion.html` en tu navegador web (haz doble clic o clic derecho y elige "Abrir con" → un navegador).
- No se necesita un servidor ni un proceso de compilación para la demostración básica: la aplicación se ejecuta directamente desde el sistema de archivos.
